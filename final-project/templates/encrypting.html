What Is Fernet?

Fernet is a specification for symmetric encryption with an authentication method, symmetric meaning it uses the same key to both encrypt and decrpyt data, authentication meaning it provides an integrity check that the encrpyted data has not been modified. It uses AES-128 in CBC mode; for authentication, it uses HMAC-SHA256 for message authentication.

How Secure Is It?

In terms of security, it's considered secure when used correctly. Fernet relies heavily on securely managing your encryption keys, never store keys in plain text. It's good for smaller pieces of data, using Fernet for very large data is not ideal.
